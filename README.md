# 🏫 VIT Chennai 7th Floor – HCI Project (3D Virtual Tour)

## 📌 Overview
This project is a **Human-Computer Interaction (HCI)** based application that provides an interactive 3D virtual tour of the 7th floor at VIT Chennai using the Matterport platform.

It allows users to explore a real-world indoor environment digitally, improving accessibility, spatial understanding, and user experience.

---

## 🎯 Objectives
- Create an immersive virtual navigation system  
- Enhance user interaction with real-world spaces  
- Apply HCI principles in spatial visualization  
- Improve accessibility through virtual exploration  

---

## 🚀 Features
- Interactive 3D walkthrough of VIT Chennai (7th floor)  
- Smooth navigation with real-time perspective changes  
- High-quality visual rendering  
- User-friendly interface  
- Accessible via browser

## 📦 MatterPak Integration
This project utilizes the MatterPak add-on, which provides:

- 3D model data (.OBJ)  
- Point cloud data (.XYZ)  
- Floor plan images  
- Structural spatial data
